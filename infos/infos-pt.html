kalan \kà.la~\: Verbo en lengua bambara que significa leer, aprender, enseñar.

Estamos orgullosos de apresentar o lendário Tuxmath, reescrito em Javascript/HTML5. Isso permite jogar o TuxMath (sem ter que instalá-lo), a partir de qualquer navegador da web, mas também em telefones e tablets. Esta nova versão do Tuxmath foi criada como parte do projeto Afrikalan, cujo objetivo é fazer com que o software educativo gratuito seja acessível na África ocidental. Se você gosta deste aplicativo, convidamos você a descobrir e apoiar as ações do projeto Afrikalan com escolas e bibliotecas na África Ocidental.

A versão original do TuxMath está disponível no site do Tux4Kids: http://www.tux4kids.com/tuxmath.html. La mayor parte de los elementos gráficos y sonoros proceden de esta última.

Nesta nova versão foram feitas algumas melhorias:

  • A opção "nível automático": Ao ativar esta opção, o jogo mudará automaticamente para outro nível se o aluno achar muito fácil ou muito difícil com as operações a serem resolvidas.
  • Níveis adicionados com três ou mais operações numéricas.
  • Uma penalidade (iglu destruído) em caso de muitas respostas incorretas (para que a estratégia de tentar todas as respostas não vença).
  • Possibilidade de jogar com 3 temas gráficos: "Classico", "Original" y "Afrikalan".

Este aplicativo é distribuído sob licencia GNU GPL V3, que le garantiza el derecho a usarla y compartirla libremente. Las personas que lo deseen pueden acceder al código fuente en https://gitlab.com/Afrikalan/tuxmath-js.

Como jogar:

  • Escolha o nível desejado no menu.
  • Asteroides caem do céu. Destrua-os antes que caiam no chão! Para apontar corretamente para um asteroide com o feixe de laser, resolva a operação escrita no asteroide, insira o resultado no teclado e valide com a tecla "Enter". Se a resposta estiver correta, o laser atinge e destrói o asteroide. Faça o mesmo para todos os asteróides presentes na tela. Em caso de resposta errada, o laser vai na direção errada (Cuidado com danos colaterais!).
  • A tecla TAB no teclado permite pausar o jogo.
  • O ícone
    cambia a pantalla completa.

URL direto para um determinado nível:

É possível fornecer um URL direto para um nível. (Útil para professores que desejam orientar seus alunos na revisão de uma tabela de multiplicação precisa, por exemplo.)

Para fazer isso, vá para http://www.tuxmath.org e, no menu, clique com o botão direito do mouse no nível desejado e selecione "copiar caminho do link"). Obtém uma URL no formato http://tuxmath.org/game.html?lvlId=xxxxxxxxxx, onde xxxxxxxxxx é o nível selecionado.

É possível adicionar diferentes opções ao URL:

Opção Efeito Valores possíveis
&opt_lang=X Selecione o idioma do jogo (substitua X pelo idioma desejado). X=fr français.
X=pt Portugues.
X=en anglais.
X=es espagnol.
&opt_autolevel=X Ativa ou desativa a mudança automática de nível quando o nível selecionado é muito fácil ou muito difícil para o aluno. (desativado por padrão) X=1 nível automático habilitado.
X=0 nível automático desabilitado (padrão).
&opt_theme=X Selecione o tema gráfico (substitua o X pelo tema desejado). X=Classico (por defecto).
X=original.
X=afrikalan.
&opt_osk=X Mostra ou oculta o teclado virtual na tela. X=1 Teclado visível (padrão).
X=0 Teclado oculto.

Por exemplo, a URL http://www.tuxmath.org/game.html?lvlId=mul_table_5&opt_lang=es&opt_osk=1 levará diretamente ao nível "tabela de multiplicação de 5", garantindo que o idioma selecionado seja o espanhol e que o virtual teclado está ativado.

O que é Afrikalan?

Esta versão do TuxMath foi criada como parte do projeto Afrikalan.

Afrikalan é um projeto que visa tornar o software educacional gratuito acessível a crianças nos países do sul.

Trata-se da criação de computadores especificamente concebidos para uso educativo, que são doados a estruturas públicas ou sem fins lucrativos de acolhimento de crianças: escolas, bibliotecas, abrigos, etc...

No Mali, em colaboração com a associação francesa Bilou Toguna, mais de 133 bibliotecas foram equipadas com computadores educativos. Na Guiné, 14 bibliotecas foram equipadas em colaboração com a ONG francesa Les enfants de l'Aïr. Nosso site permitirá que você conheça mais sobre as ações realizadas.

Afrikalan também é um trabalho de criação de conteúdo educacional adaptado ao contexto cultural em que esses computadores educacionais são fornecidos.

É neste contexto que esta versão do Tuxmath foi criada. Este aplicativo é distribuído sob o Licença GNU GPL V3, que concede a você o direito de usá-la e compartilhá-la livremente.

Se você achar este aplicativo útil, você pode apoiar o projeto Afrikalan para que as crianças na África Ocidental possam se beneficiar dele.
